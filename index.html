Example
rgb Red Green Blue
Defines a colour based on the red, green, and blue components. This can be expressed through hexadecimal and functional notations. The RGB numbers span from 255 to 0, where 255 indicates 100% presence of that colour.
Hexadecimal:
#RRGGBB, #RGB
Functional:
rgb(R G B)
Hexadecimal:
#7FFFD3
Functional:
rgb(127, 255, 211)
rgba Red Green Blue Alpha
RGB operates the same as above, but the Alpha element is a number from 0 to 1 that represents a percentage. If the number is set to 1 that indicates full opacity (100%), whereas if it is set to 0 that indicates full transparency (0%).
Hexadecimal:
#RRGGBB[A], #RGB[A]
Functional:
rgba(R G B, A)
Hexadecimal:
#0007
Functional:
rgba(127, 255, 211, 0.7)
